## One-Liner series: Bun

[source: Bun docs](https://bun.com/docs/installation)

### Install on Linux and macOS

```bash
curl -fsSL https://bun.com/install | bash
```

### Install on Windows (PowerShell)

```powershell
powershell -c "irm bun.sh/install.ps1|iex"
```

### Update

```bash
bun upgrade
```
