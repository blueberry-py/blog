## One-Liner series: juliaup

[source: julialang.org](https://julialang.org/downloads/#install_julia)

### Install on Ubuntu

```bash
curl -fsSL https://install.julialang.org | sh
```

### Update on Ubuntu (via 'release' channel)

```bash
juliaup update release
```
